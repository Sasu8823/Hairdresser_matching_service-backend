美容師マッチングサービス 外注依頼用資料
LINE × Stripe 成果課金＋顧客ランク制マッチング構想

プロジェクトの目的

・美容師がLINE経由でお客様とマッチングできる仕組みを構築
・マッチング1件につき500円の成果課金（Stripeで自動決済）
・お客様には施術予算に応じたランク（例：S〜C）を設定し、マッチング精度を向上

サービス全体構成（概要）

■ 使用ツール：
- LINE公式アカウント（お客様受付）
- Googleフォーム（施術希望入力）
- Stripe（成果課金500円決済）
- スプレッドシート（履歴・管理）

■ 流れ：
① 顧客が施術希望をLINEで投稿
② 美容師が応募・顧客が選定
③ 有料案件はStripeで500円支払い（無料案件は対象外）
④ 評価・リピートに繋げる

顧客ランク制度の導入

■ 背景：
美容師が単価の低い案件ばかり選ぶと利益率が悪化する

■ ランク例：
- Sランク：予算 ¥8,000以上＋リピート希望
- Aランク：予算 ¥5,000〜¥8,000
- Bランク：モデル施術（¥1,000〜¥5,000）
- Cランク：完全無料案件

■ マッチング優先度や報酬アップ制度に活用可能


外注範囲のご依頼内容

■ LINE連携：
- リッチメニュー・フォーム自動応答構築
- 顧客条件とランク入力フォーム

■ Stripe構築：
- 商品登録（500円）・成果課金リンク作成
- ダッシュボード通知設定

■ Googleスプレッドシート：
- マッチング履歴・支払い記録管理用

■ その他：
- 美容師／顧客向けメッセージテンプレ作成
